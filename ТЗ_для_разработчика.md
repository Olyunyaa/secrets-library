# Онбординг + Библиотека для основного бота Секретов

## Коротко

Мы сделали рабочий прототип онбординга для бота Секретов. Нужно перенести эту логику в основной бот Тани. Весь код — в `bot.py`, можно потыкать: [@SecretsLibraryBot](https://t.me/SecretsLibraryBot).

**Репо:** `github.com:Olyunyaa/secrets-library.git` — там весь код и все данные.

---

## Сценарий пользователя

1. Пользователь нажимает `/start`
2. Бот задаёт **4 вопроса** (inline-кнопки, мульти-выбор в Q1 и Q2):
   - Q1: Кто вы? (5 вариантов)
   - Q2: На чём фокус? (11 вариантов → маппятся на 11 pain points в roadmap)
   - Q3: Период дорожной карты (7 / 14 / 28 / 60 дней)
   - Q4: Как получать? (всё сразу / drip / drip + напоминания)
3. Бот собирает посты из `roadmap_all_pains_v4.json` по выбранным темам
4. Отправляет материалы через **кнопку Mini App** (библиотека `app.html` с `?ids=...`)
5. Если выбран drip — батчи по 2 поста каждые 1-4 дня (зависит от периода), каждые 3 батча спрашивает фидбэк
6. После онбординга пользователь может написать **свободный текст** → Claude подбирает посты по теме и отправляет кнопку Mini App

## Что нужно от Вовы

### Вариант А: взять bot.py целиком
`bot.py` — полностью рабочий бот на `python-telegram-bot 21.*` + `anthropic`. Можно вырезать из него нужные части и вставить в основной бот.

### Вариант Б: интегрировать по спеке
Ключевые куски логики в `bot.py`:
- **Онбординг** — `ConversationHandler` с состояниями Q1→Q2→Q3→Q4→CHAT (строки ~490-690)
- **Drip-рассылка** — `drip_delivery_job()`, ежедневная задача в JobQueue (строки ~730-860)
- **Свободный чат** — `classify_request()` + `generate_selection()`, двухэтапный поиск через Claude (строки ~340-450)
- **Кнопки «Больше про тему»** — `handle_more_topic()` (строки ~880-960)
- **Коррекция маршрута** — `adjust_remaining_posts()` через Claude (строки ~450-510)

## Файлы данных

| Файл | Что это | Где используется |
|------|---------|-----------------|
| `roadmap_all_pains_v4.json` | 11 pain points, для каждого список постов | Формирование дорожной карты |
| `knowledge_base.js` | База 496 постов (2.3 MB) | Mini App + свободный поиск через Claude |
| `app.html` | Mini App — библиотека контента | Открывается из Telegram по кнопке |

Mini App хостится на GitHub Pages: `https://olyunyaa.github.io/secrets-library/app.html`
Фильтрация: `?ids=channelId_messageId,channelId_messageId,...`

---

## Зависимости

- **Claude API** (Anthropic) — модель `claude-haiku-4-5-20251001` для подбора постов, коррекции, свободного чата. Нужен `ANTHROPIC_API_KEY`. Стоимость ~0.01$ на запрос.
- **python-telegram-bot[job-queue]==21.*** — для inline-кнопок, ConversationHandler, JobQueue
- **Mini App** — `app.html` + `knowledge_base.js` на любом статик-хостинге

---

## Важно: формат ID постов

Mini App ищет посты по ключу `{channel_id}_{message_id}` (например `2030927165_17`). Roadmap использует тот же формат. При передаче ID в Claude и в URL Mini App — именно этот формат.
